# Sora Vault
Luxury collection management app for watches, sneakers, and purses.

## Features
- Multi-collection support (watches, sneakers, purses)
- Collection tracking with type-specific specifications
- Wear/usage analytics
- Financial valuation
- Trips & events tracking
- Water usage tracking (for watches)
- Wishlist with AI recommendations
- Trade matching between collectors
- Community messaging & forum
- Admin panel

## Collection Types

### Watches
- Movement, case material, crystal, water resistance
- Power reserve, lug-to-lug measurements
- Warranty tracking, rarity, and historical significance

### Sneakers
- Colorway, size (US/UK/EU/CM), SKU, style code
- Condition (deadstock, VNDS, used, worn)
- Collaboration, release date, silhouette

### Purses
- Material, hardware color, size category
- Closure type, strap type
- Authenticity verification, serial number

## Tech Stack
React + TypeScript + Vite + Tailwind + Supabase

## Install
npm install
npm run dev
