## Managing Collections

### Creating a New Collection

1. Click the **collection dropdown** in the top navigation
2. Select **"Create New Collection"**
3. Enter a name (e.g., "Everyday Watches", "Sneaker Rotation")
4. Choose your collection type:
   - **Watches** ‚åö - For timepieces
   - **Sneakers** üëü - For shoes
   - **Purses** üëú - For handbags
5. Click **"Create Collection"**

### Switching Between Collections

Use the collection dropdown in the navigation bar to switch between your collections. The entire interface will update to match the selected collection type.

### Sharing Collections

Collections can be shared with other users:

1. Open **Collection Settings**
2. Click **"Share Collection"**
3. Enter the user's email
4. Choose their permission level:
   - **Viewer**: Can view items only
   - **Editor**: Can add and edit items
   - **Owner**: Full control (you can have co-owners!)
5. Send invite

## Adding Items

### Quick Add

The fastest way to add an item:

1. Click the **"+"** or **"Add [Item Type]"** button
2. Fill in required fields:
   - Brand (e.g., "Rolex", "Nike", "Louis Vuitton")
   - Model (e.g., "Submariner", "Air Jordan 1", "Speedy 30")
   - Purchase cost
3. Optionally add:
   - Purchase date
   - MSRP
   - Photos
   - Type-specific details
4. Click **"Save"**

### Detailed Add

For comprehensive tracking:

1. Use the **"Add [Item Type]"** button
2. Fill in the **Basic Information** tab
3. Navigate to **Specifications** tab for type-specific fields
4. Add photos in the **Photos** tab:
   - Upload images (supports drag-and-drop)
   - Use AI image enhancement (optional)
5. Set **Warranty** information (if applicable)
6. Configure **Trade Availability** if you're open to trades
7. Save your item

### Uploading Photos

- **Supported formats**: JPG, PNG, HEIC
- **Max size**: 10MB per photo
- **AI Enhancement**: Automatically improves photo quality
- **Multiple photos**: Add front, side, detail shots

## Tracking Usage

### Daily Wear Logging

Track when you wear/carry your items:

1. Click **"Log Wear"** from the dashboard
2. Select the item
3. Enter the date and number of days
4. Add optional notes
5. Save

**Quick Log**: Use the quick-add button to log today's wear for frequently used items.

### Usage Statistics

View detailed usage stats:
- **Total days worn/carried**
- **Last worn date**
- **Wear frequency** (daily, weekly, monthly averages)
- **Cost per wear** (purchase cost √∑ total days worn)
- **Most worn items**
- **Least worn items** (candidates for selling?)

### Trips and Events

For watches and sneakers, associate items with trips and events:

1. Navigate to **Trips** or **Events**
2. Create a new trip/event
3. Link items that you brought/wore
4. Track total days for each item during the trip

## Tips & Best Practices

### For Watch Collectors

‚úÖ **Do:**
- Log movement type accurately (affects servicing needs)
- Track water resistance ratings
- Document service history
- Upload warranty cards
- Record purchase receipts for authenticity

‚ùå **Don't:**
- Forget to log wear for accurate cost-per-wear
- Ignore warranty expirations
- Skip photos (crucial for insurance/resale)

### For Sneaker Collectors

‚úÖ **Do:**
- Record exact size and region (US/UK/EU)
- Keep boxes and tags for deadstock items
- Document colorway and style codes
- Track release dates for limited editions
- Log condition changes over time

‚ùå **Don't:**
- Mix up sizes between regions
- Forget to note OG all status
- Lose track of collaboration details
- Skip SKU/style code (crucial for verification)

### For Purse Collectors

‚úÖ **Do:**
- Always verify authenticity if possible
- Record serial numbers
- Keep dust bags and authenticity cards
- Document material and hardware
- Track condition meticulously
- Upload detailed photos of hardware, stitching, and stamps

‚ùå **Don't:**
- Skip serial number documentation
- Forget to note included accessories
- Ignore condition changes
- Mix up material types

### General Best Practices

1. **Regular Updates**: Log wear/usage weekly
2. **Photo Quality**: Use good lighting for photos
3. **Accurate Data**: Be honest about condition and details
4. **Backup**: Export your data monthly
5. **Insurance**: Use collection value reports for insurance
6. **Security**: Enable 2FA on your account
7. **Privacy**: Adjust privacy settings for public collections

## Troubleshooting

### Common Issues

**Q: I don't see my collection type option**
A: Ensure you're on the latest version. Refresh your browser or update the app.

**Q: Photos won't upload**
A: Check file size (max 10MB) and format (JPG, PNG, HEIC only).

**Q: Wear logs not saving**
A: Ensure dates are valid and items are active (not sold/traded).

**Q: Can't switch collections**
A: Verify you have at least one collection. Try refreshing the page.
